# Game component references
## Description of this
I will refer to various parts of the game by these phrases. 
Consider them while responding to my prompts.
I might use only parts of it, or slightly different wordings
We will keep this list updated, to help each other's work

## Component references
- link shared game - when game was loaded from a shared game url
- solo game - when game was initiated by player
- solve phase - when the user is doing the word matching
- config phase - when there is any configuration modal open and the game is not yet started
- shared game welcome message - when user loads a shared game, this should be displayed
